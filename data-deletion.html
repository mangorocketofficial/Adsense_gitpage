데이터 삭제 요청 안내

최종 수정일: 2026년 3월 31일

This page explains how users can request deletion of data related to the Thohago service.

1. 요청 방법

Thohago 관련 개인정보 또는 운영 데이터를 삭제하고 싶으신 경우 아래 이메일로 요청해 주세요. 본인 확인과 요청 범위 확인 후 삭제 절차를 진행합니다.

삭제 요청 이메일: mangorocket.official@gmail.com

메일 제목 예시: [Thohago] Data Deletion Request

2. 함께 보내주시면 좋은 정보

  1. 이름 또는 담당자명
  2. 매장명 또는 서비스 이용 식별 정보
  3. 연결된 이메일 주소 또는 연락 가능한 정보
  4. Telegram chat ID 또는 운영 채널 식별 정보가 있다면 함께 제공
  5. Meta 플랫폼을 통해 요청한 경우 관련 계정 정보 또는 요청 식별 정보

3. 삭제 대상 범위

  • 운영 채널 식별 정보 및 연결 정보
  • 업로드된 사진, 영상, 음성 메모, 텍스트 입력
  • 전사 결과, 생성 초안, 게시 준비 자료, 운영 로그
  • 지원 문의 및 요청 처리 과정에서 수집된 관련 정보

단, 법령 준수, 분쟁 대응, 보안 기록 유지 등으로 인해 일부 정보는 필요한 범위 내에서 일정 기간 보관될 수 있습니다.

4. 처리 절차

접수

이메일로 삭제 요청을 접수합니다.

확인

필요 시 본인 확인 또는 추가 식별 정보를 요청할 수 있습니다.

처리

일반적으로 7영업일 이내 1차 회신, 특별한 사정이 없으면 30일 이내 처리 완료를 목표로 합니다.

5. 요청 템플릿

예시

Name / Store Name:

Contact Email:

Telegram Chat ID or Service Identifier:

Related Meta Account Information (if applicable):

Request: Please delete my data associated with the Thohago service.

6. 문의처

삭제 요청 진행 상태나 추가 문의가 필요한 경우 아래 연락처로 문의해 주세요.
