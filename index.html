02

엑셀 주제 불러오기, 자동 글쓰기, 발행 흐름을 Windows 데스크톱에서 다룹니다.

03

결제, 소개, 다운로드를 한 페이지에

구매 전 설명 확인, 결제 문의, 제품 다운로드, 지원 문서 접근을 한 번에 정리합니다.

ACCOUNT FLOW

결제와 사용 방식

이 페이지의 핵심은 “웹 결제 계정 = 제품 로그인 계정”입니다. 결제 후 제품에서 다른 구글 계정으로 로그인하면 권한 확인이 맞지 않을 수 있으므로 같은 계정을 유지하는 흐름을 전면에 배치했습니다.

STEP 1

구글 계정으로 결제 진행

웹페이지에서 결제를 진행할 때 기준이 되는 구글 계정을 먼저 확정합니다.

STEP 2

결제 정보와 권한 등록

결제 완료 후 해당 구글 계정 기준으로 사용 권한을 연결하는 구조를 안내합니다.

STEP 3

제품 다운로드 및 설치

다운로드한 Windows 데스크톱 앱을 설치한 뒤 첫 실행 준비를 합니다.

STEP 4

동일한 구글 계정으로 로그인

제품에서 같은 구글 아이디로 로그인해야 결제 정보를 기준으로 사용이 연결됩니다.

중요

웹에서 결제한 구글 계정과 제품에서 로그인하는 구글 계정은 반드시 동일해야 합니다. 이 안내는 메인 랜딩, 요금제, 다운로드 섹션 전체에서 반복적으로 보이도록 구성했습니다.

SERVICE

또하고 네이버 자동화 글쓰기 서비스 소개

실제 제품 프로젝트 기준으로 보면 데스크톱 앱 안에서 일반 글쓰기, 브랜드커넥트 글쓰기, 쿠팡 링크 기반 글쓰기, 엑셀 주제 관리 흐름이 이미 나뉘어 있습니다. 이 페이지는 그 구조를 사용자에게 이해하기 쉽게 압축해서 설명합니다.

GENERAL WRITING

엑셀 주제 불러오기와 자동 글쓰기

엑셀 파일에서 주제를 불러온 뒤 초안을 생성하고, 순서대로 자동 글쓰기를 실행하는 흐름을 사용할 수 있습니다.

BRAND CONNECT

브랜드커넥트 링크 기반 글 작성

브랜드커넥트 링크를 기준으로 상품 정보를 스크랩하고 초안과 이미지 흐름을 연결하는 작업을 제품 안에서 처리합니다.

COUPANG

쿠팡 링크 스크랩과 발행 준비

쿠팡 링크나 파트너스 링크를 가져와 자동 초안을 생성하고, 수정 후 발행까지 이어지는 데스크톱 작업 흐름을 다룹니다.

PROMPT + EDIT

프롬프트 수정과 직접 편집

자동 생성만 끝이 아니라 제목, 본문, 태그, 프롬프트를 직접 수정한 뒤 발행 흐름을 조정할 수 있도록 설계되어 있습니다.

제품 다운로드

이 섹션은 실제 다운로드 버튼을 연결하기 위한 자리입니다. 지금은 사용자가 설치 전 확인해야 하는 계정 기준과 다운로드 안내를 먼저 정리했습니다.

Windows 데스크톱 앱

결제 완료 후 사용하는 프로그램은 Windows 데스크톱 앱 기준입니다. 앱 실행 뒤에는 웹에서 결제한 것과 같은 구글 아이디로 로그인해야 사용 권한이 이어집니다.

설치 전 체크리스트

  1. 결제에 사용한 구글 계정을 확인합니다.
  2. 다운로드 받은 Windows 앱을 설치합니다.
  3. 첫 실행 후 같은 구글 계정으로 로그인합니다.
  4. 권한 확인 후 네이버 자동화 글쓰기를 시작합니다.
